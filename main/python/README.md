## How to Run

- Use Python 3 to run any script, for example:
  - `python3 phase_1_python/ex.2.py`
  - `python3 phase-2-python/chapter_4/01_lists.py`
- Open any file in a code editor to view examples and learn the concept behind each script.
- The root README gives the repository overview, while each phase folder contains more detailed learning notes.
