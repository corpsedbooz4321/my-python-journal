# Chapter 4: Lists, Tuples, and Practice Exercises

This folder contains Python practice code focused on Chapter 4 concepts: list operations, tuple behavior, and related practice problems.

## What’s inside

- `list.py`
  - Basic list usage and mutability
  - Accessing items by index
  - Updating list elements
  - Slicing lists

- `02_list_method.py`
  - Common list methods:
    - `append()`
    - `sort()`
    - `reverse()`
    - `insert()`
    - `pop()`
    - `remove()`
  - Demonstrates how lists can be changed after creation

- `03_tuples.py`
  - Tuple creation and immutability
  - Shows that tuple elements cannot be reassigned

- `04_tuples_methods.py`
  - Tuple methods like `count()` and `index()`
  - Built-in functions that work with tuples:
    - `len()`
    - `max()`
    - `min()`
    - `sum()`
  - Tuple slicing and concatenation examples

- `practiceset.py/`
  - `01_problems.py`: build a list of seven fruits from user input
  - `02_proble.py`: collect and sort seven marks from user input
  - `03_practice.py`: verify tuple immutability with item assignment
  - `04_problem.py`: sum a list of numbers, including zero values
  - `05_problem.py`: count how many zeros are in a list

## Learning focus

- Understanding the difference between mutable lists and immutable tuples
- Using list methods to modify collections
- Applying tuple methods and built-in functions for tuple analysis
- Practicing user input collection and list manipulation
- Working with list summarization tasks like sum and counting values

## How to use these files

1. Open a file in this folder.
2. Run it with Python to see the behavior of list or tuple operations.
3. Modify values or add new examples to deepen understanding.

## Notes

- Lists support element assignment and many modification methods.
- Tuples do not allow item assignment after creation.
- The practice set reinforces input handling, sorting, summation, and counting.
